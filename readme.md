# Cursor - AI 驱动的智能代码编辑器

## 产品概述

Cursor 是一款由 Anysphere 开发的 AI 驱动代码编辑器，旨在显著提升开发者的编程效率。它基于 VS Code 的开源编辑器底层构建，集成了先进的大语言模型（LLMs），提供智能代码补全、上下文感知、自动化操作等强大功能，让编码、调试、重构等开发任务更加高效直观。

---

## 核心功能

### 1. 智能代码补全（Autocomplete）

- **整行/整段预测**：不只是简单的单词补全，能够预测整行甚至整段代码
- **Diff 预览**：实时显示代码的增删改部分，让您清楚看到即将进行的修改
- **上下文感知**：基于当前代码上下文和项目结构，提供精准的代码建议
- **预测下一个编辑位置**：智能预测您接下来可能编辑的位置，提升编码流畅度

### 2. Agent 模式（自动化任务执行）

- **跨文件代码生成**：自动在多个文件中生成相关代码，保持代码一致性
- **终端命令执行**：自动执行终端命令，完成构建、测试、部署等任务
- **自动修复 Lint 错误**：检测并自动修复代码中的 lint 提示和错误
- **用户控制权**：所有自动执行的命令都需要用户确认，确保安全性

### 3. Chat 聊天助手

- **代码上下文理解**：理解当前文件、选定代码块或整个项目的上下文
- **自然语言交互**：支持用自然语言提问、解释代码、诊断错误
- **代码解释**：帮助理解复杂代码逻辑和架构设计
- **错误诊断**：快速定位问题并提供修复建议
- **图像上下文**：支持上传截图作为上下文，实现更精准的代码理解

### 4. @ 引用系统

- **文件引用**：使用 `@文件名` 引用特定文件，为 AI 提供精确上下文
- **文件夹引用**：引用整个文件夹，让 AI 理解模块结构
- **代码片段引用**：引用特定代码片段，进行精准的代码分析
- **文档引用**：引用项目文档，确保 AI 遵循项目规范
- **历史对话引用**：引用之前的聊天内容，保持对话连贯性

### 5. 外部工具和上下文集成

- **目录结构读取**：自动读取和分析项目目录结构
- **全文检索**：在代码库中进行全文搜索，快速定位相关代码
- **关键字/模式搜索**：支持正则表达式和模式匹配搜索
- **公共文档库集成**：集成主流框架和库的官方文档
- **自定义文档**：支持添加项目特定的文档和规范
- **网页信息检索**：通过 MCP（Model Context Protocol）连接第三方服务
- **第三方服务集成**：支持 Slack、Drive 等外部工具集成

### 6. 终端命令支持

- **自然语言转命令**：用自然语言描述终端操作，自动生成对应命令
- **命令执行确认**：在执行命令前提供确认机制，确保安全性
- **自动化工作流**：在 Agent 模式下自动执行命令序列

### 7. 多语言和框架支持

- **主流编程语言**：支持 JavaScript、TypeScript、Python、Java、Rust、Go、C++ 等
- **前端框架**：支持 React、Next.js、Vue、Angular、Svelte 等
- **后端框架**：支持 Express、Django、Flask、Spring Boot 等
- **移动开发**：支持 React Native、Flutter 等
- **数据科学**：支持 Jupyter Notebook、数据分析工具等

### 8. 隐私安全与企业特性

- **隐私模式（Privacy Mode）**：启用后，您的代码不会用于模型训练，不会长时间保留
- **企业订阅**：支持团队管理、统一账单、SSO 单点登录
- **组织设定**：支持组织级别的配置和策略管理
- **访问控制**：细粒度的权限管理，确保代码安全

## 技术特性

### 长上下文支持与项目感知

- **全局代码库理解**：不仅理解当前打开的文件，还能理解整个代码库的架构
- **依赖关系分析**：自动分析模块之间的依赖和组件关系
- **跨文件重构**：支持大规模跨文件的代码重构，保持一致性
- **架构感知**：理解项目的整体架构设计模式

### 多模型支持

- **多种 AI 模型**：支持 OpenAI、Anthropic、Google 等主流 AI 模型
- **Cursor 专用模型**：提供 "cursor-small" 和 "Composer" 等优化模型
- **自定义模型**：允许用户插入自己的 API key 或模型（根据订阅计划）
- **模型切换**：可以根据不同场景选择最适合的模型

### 性能优化

- **快速响应**：代码补全和编辑预测响应速度快，实时反馈
- **大型代码库支持**：在大型项目中保持流畅的使用体验
- **资源优化**：智能管理计算资源，确保编辑器性能

### VS Code 兼容性

- **无缝迁移**：基于 VS Code 开源版本，支持导入 VS Code 配置
- **主题支持**：兼容 VS Code 的主题和配色方案
- **插件兼容**：支持大部分 VS Code 插件和扩展
- **快捷键**：保持 VS Code 的快捷键习惯，降低学习成本

## 工作流程示例

### 典型开发流程

1. **需求分析**：通过 Chat 与 AI 讨论功能需求
2. **代码生成**：使用 Agent 模式自动生成代码框架
3. **代码补全**：在编写过程中获得智能补全建议
4. **错误检测**：自动检测并修复 lint 错误
5. **测试执行**：自动运行测试命令验证功能
6. **代码审查**：使用 Chat 进行代码审查和优化建议

### 重构工作流

1. **分析影响范围**：使用 @ 引用系统分析需要修改的文件
2. **生成重构方案**：通过 Chat 讨论重构策略
3. **执行重构**：Agent 模式自动执行跨文件重构
4. **验证一致性**：自动检查重构后的代码一致性
5. **运行测试**：确保重构后功能正常

## 注意事项

### 使用建议
- **人工校对**：在复杂边缘用例中，建议人工审查 AI 生成的代码
- **谨慎使用自动执行**：在 Agent 模式下执行命令时，注意确认操作的安全性
- **理解上下文**：确保 AI 理解正确的项目上下文，使用 @ 引用系统提供精确信息

### 限制说明
- **模型依赖**：某些功能依赖于第三方 AI 模型或服务的可用性
- **复杂场景**：在非常复杂或罕见的用例中，建议可能需要人工调整
- **学习曲线**：充分利用所有功能需要一定的学习和适应时间
